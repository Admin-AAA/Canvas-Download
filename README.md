# SJTU Canvas Video Downloader
![143601766394163_ pic](https://github.com/user-attachments/assets/3f66dae1-1d64-458b-8115-8171c7e76fd1)

这是一个用于从 SJTU Canvas 平台下载教学视频的浏览器脚本工具。

## 功能特点

- **自动嗅探**：自动检测页面上的 MP4 视频资源。
- **绕过 403 限制**：通过在浏览器端直接请求并构建 Blob 对象，有效解决因防盗链（Referer）导致的下载失败问题。
- **无需安装**：纯 JavaScript 代码，直接在浏览器控制台运行。
- **备用方案**：如果直接下载失败，自动生成带有 Cookie 和 Header 的 cURL 命令。

## 使用方法

1. **打开视频页面**
   在浏览器中打开包含目标视频的 Canvas 课程页面。

2. **打开开发者工具**
   - Windows: 按 `F12` 或 `Ctrl + Shift + I`
   - Mac: 按 `Cmd + Option + I`

3. **运行脚本**
   - 切换到 **Console (控制台)** 标签页。
   - 复制 `download_sjtu_video.js` 文件中的所有代码。
   - 粘贴到控制台并按 **Enter** 回车。

4. **下载视频**
   - 脚本运行后，页面右上角会出现悬浮面板。
   - 点击 **"播放"** 按钮让视频开始缓冲（如果列表未显示）。
   - 点击 **"🚀 尝试直接下载"** 按钮即可保存视频。

## 注意事项

- 本脚本仅供学习交流使用。
- 请勿传播下载的课程资源。
